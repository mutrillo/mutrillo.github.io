Política de Privacidade

Última atualização: 25 de fevereiro de 2026

1. Introdução

O Achados Shopee ("nós", "app") respeita sua privacidade. Esta política explica como coletamos, usamos e protegemos seus dados ao usar o aplicativo integrado com a API do TikTok.

2. Informações que coletamos

  • Dados da conta TikTok (com seu consentimento): ID de usuário, nome de usuário, avatar, access token (temporário).
  • Informações de produtos Shopee que você seleciona manualmente.
  • Dados de uso do app (logs anônimos para melhorar o serviço).

3. Como usamos seus dados

Usamos apenas para:

  • Autenticar e postar vídeos no seu TikTok (via API oficial).
  • Gerar sugestões de conteúdo.
  • Melhorar o app (sem vender dados a terceiros).

4. Compartilhamento de dados

Não vendemos nem compartilhamos seus dados pessoais. Compartilhamos apenas com o TikTok (via API) o necessário para o funcionamento.

5. Segurança

Usamos medidas de segurança padrão. Tokens de acesso são armazenados temporariamente e criptografados.

6. Seus direitos

Você pode revogar acesso ao app nas configurações do TikTok a qualquer momento.
